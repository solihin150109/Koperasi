# Frontend Sistem Informasi Koperasi Jambi

This contains the frontend of Jambi Class I TPI Immigration Office Workers Cooperative Information System.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
